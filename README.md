Growth Homepage
===

这个项目用于[https://github.com/phodal/growth](https://github.com/phodal/growth)的首页。

Growth是一款专注于Web开发者成长的应用，涵盖Web开发的流程及技术栈，Web开发的学习路线、成长衡量等各方面。在这里，你将通过不断检验自己的学习成效，形成属于你自己的独一无二的个性技能图谱。
